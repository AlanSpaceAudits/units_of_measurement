# Units of Measurement

**A System of Mutually Reliant Ratios Creates the Internal Consistency of the System**

Aether Cosmology Research Group.

The brief derives the distance to the Moon, the distance to the Sun (one AU),
and the speed of light twice over: once in the Chinese *li*, once in the
metric *metre*. It then converts between the two unit systems and shows every
value matches. It also covers the aberration route to the AU and the 1983
redefinition of the metre through *c*.

## Files

- `units_of_measurement.pdf`: the brief (6 pages)
- `units_of_measurement.tex`: LaTeX source
- `units_of_measurement.ipynb`: companion notebook with the worked computations
- `null_hypothesis.md`: the internal-consistency null hypothesis (H₀ / H₁)
- `ac_logo.png`: figure used in the brief

## Build

The PDF is compiled with `pdflatex`:

```
pdflatex units_of_measurement.tex
```

The notebook runs under any standard Python 3 environment; the computations
use only the standard library.
